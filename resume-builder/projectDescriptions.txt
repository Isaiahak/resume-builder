Stock Monitoring Tool

Developed a locally hosted stock monitoring application that allows users to track selected stocks and receive relevant news during significant price movements. Built a TypeScript-based frontend for managing watchlists and displaying real-time insights. Implemented scheduled web scraping jobs to collect daily stock data and trigger event-based updates. Designed a word-association embedding approach to map financial news articles to relevant stocks using sector-based keyword groupings. Optimized article retrieval and processing pipelines to reduce latency and improve relevance. Built in Go to support future cloud deployment and scalable backend processing.

Optimization Tool

Developed a performance analysis tool that processes 30-minute gameplay recordings and generates concise optimization insights within seconds. Implemented additional computer vision techniques to identify UI elements and icons for contextual analysis. Built a React and TypeScript frontend for visualizing insights and summaries. Deployed a cloud-based backend leveraging GPU acceleration to efficiently process compute-intensive ML workloads.

Movie Recommendation Website

Built a full-stack movie recommendation platform that generates personalized suggestions based on user ratings and preferences. Developed a React frontend for user interaction and rating workflows, and a FastAPI backend to handle recommendation logic and data processing. Designed a recommendation system using vector embeddings stored in MongoDB to model user preferences. Implemented data preprocessing pipelines to clean and transform user input into initial embeddings, addressing cold-start scenarios for new users. Scheduled periodic updates via cron jobs to refresh recommendations and improve relevance over time. Integrated custom user authentication to support personalized experiences.
